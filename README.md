# Обновление остатков и цен для Ozon

Этот скрипт автоматически обновляет товары на Ozon на основе остатков и
цен поставщика.

## Что делает скрипт

-   Получает список товаров из Ozon
-   Загружает файл остатков от поставщика (Casio / Timeworld)
-   Сравнивает товары по артикулу (код товара)
-   Обновляет на Ozon:
    -   Количество товаров на складе
    -   Цены товаров

Если товара больше нет у поставщика - на Ozon ставится количество
**0**.

## Для кого

Для менеджеров маркетплейсов и владельцев магазинов:\
скрипт помогает не делать обновления вручную и не ошибаться в
ценах/остатках.

## Пояснение простыми словами

Есть список товаров на Ozon и список от поставщика.\
Скрипт сверяет их по артикулу и обновляет данные на Ozon, чтобы всё было
актуально.

# Обновление остатков и цен для Яндекс.Маркета

Этот скрипт помогает автоматически поддерживать актуальные данные о товарах на **Яндекс.Маркете**.  
Он берет свежие данные у поставщика и обновляет их в личном кабинете продавца.

## Что делает скрипт

- Получает список всех товаров с Яндекс.Маркета  
- Скачивает новый файл с остатками и ценами от поставщика (Casio / Timeworld)  
- Сопоставляет товары по артикулу (код товара)  
- Обновляет на Маркете:
  - Количество товара на складе  
  - Актуальные цены  

Если товара больше нет у поставщика — на Яндекс.Маркете ставится количество **0**, чтобы он не отображался как доступный.

## Для кого

Для менеджеров маркетплейсов и владельцев магазинов, которые хотят:  
- Избежать ошибок при обновлении цен и остатков  
- Сэкономить время на ручных обновлениях  
- Быть уверенными, что данные на Маркете всегда точные и свежие  

## Пояснение простыми словами

Есть два списка: товары на Яндекс.Маркете и товары у поставщика.  
Скрипт сравнивает их по артикулу и обновляет данные на Маркете, чтобы всё совпадало и покупатели видели только актуальные предложения.
